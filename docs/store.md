# Full Initiative Tracker

A feature-rich initiative tracker for Owlbear Rodeo with combat effects and smart turn management.

## Features

- **Combat Start Effect** — Full-screen animation when combat begins, synced to all players
- **Image-based Initiative List** — Token images displayed as full-width backgrounds with center crop
- **Camera Focus** — Click any entry or switch turns to auto-focus the camera on that token
- **GM Controls** — Start Combat, Previous/Next Turn, End Combat (GM only)
- **Hidden Item Support** — Hidden tokens are automatically skipped and return when made visible
- **Multi-select** — Add multiple tokens to initiative at once via box select + right click
- **Auto-detect New Tokens** — During combat, newly added tokens prompt for initiative input
- **Language Switching** — Toggle between English and Chinese (中文) in the header
- **Round Counter** — Automatic round tracking with visual badge
- **Dynamic Sorting** — Initiative values can be edited by anyone, list re-sorts automatically

## How to Use

1. Right-click a token → "Add to Initiative"
2. Set initiative values by clicking the number on each entry
3. GM clicks "Start Combat" to begin — all players see the combat effect
4. Use Previous/Next buttons to advance turns
5. Click any entry to focus the camera on that token
